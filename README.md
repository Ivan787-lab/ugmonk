# ugmonk
Landing page для тренировки навыков верстки
 
 ## Что используется в проекте?
 
 - Сборщик проектов Gulp
   - browser-sync 
   - scss-formatter
   - cache (для сжатия картинок и файлов)
   - autoprefixer
- Препроцессор scss
- git
